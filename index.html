Nokia

My journey has just begun. I joined Nokia's Mobile Networks Baseband & SoC Digital Pre-Distortion Team in July 2023 as a Senior Software Engineer.

Responsibilities

  • Spearheaded the development and maintenance of DevOps and Continuous Integration/Continuous Development (CI/CD) functionalities for enhancing digital pre-distortion (DPD) and added support for software version reporting, artifactory management, Groovy and bash scripting, Jenkins pipeline maintenance, Gerrit and Zuul integration using YAML, Jenkins shared libraries management and Docker containerization.
  • Resolved performance issues using static/dynamic analysis, C/C++ optimizations, and improved Coverity issues through C++ static code analysis tools, Coverity, C++ rule of 5 principles, and performance optimizations.
  • Integrated team code quality tools into code integration and releases using Jenkins, Clang-format, Clang-tidy, GCC sanitizer, static vs dynamic analysis tools, and metrics such as line and branch coverages for adherence to standard industry code styles like Clang, and monitored code line and branch coverages.
  • Led the integration of legacy unit tests for cross-compiler targets into the data management platform using emulators, various testing frameworks, QEMU, Cmocka, Unity testing framework, JUnit, and Splunk, driving comprehensive testing strategies and efficient testing and validation.

Ericsson

I joined Ericsson's Product Development Team in April 2021 as an Embedded Test Software Development Engineer I and progressed to the role of Embedded Test Software Development Engineer II through a promotion.

Responsibilities

  • Developed embedded software functions, including command logic for downlink radio carriers and utilities for database access. Parsed u-boot messages and Kernel ring buffer (dmesg) to develop new commands to determine the flash type.
  • Verified software using the ETSW runtime environment under Linux based on Yocto and programmed in C, C++, and Python under Version Control System Git. Addressed various improvements such as compression of flashed software into module containers
  • Implemented test cases to verify calibration data such as PA bias, VGA dynamic range, TOR step size, attenuation, TOR gain, and frequency, utilizing C++ STL containers such as vectors, and maps. Also, facilitated TOR (Transmitter Observer Receiver) basic functionalities such as checking noise level, range of gain table, ADC headroom over complete TX band, maximum and minimum gain for all power classes, and calibration temperature. Additionally, contributed to VGA (Voltage Gain Amplifier) basic functionalities like dynamic range of gain table, DSA (Digital Step Attenuators), and calibration temperature.
  • Implemented CICD using Jenkins pipelines and Python. Developed Jenkins pipeline for uploading binaries and artifacts to HTML storage sites, integrated machine-readable components into pipelines by parsing repositories of different radios, and developed interface for resigning revoked artifacts.
  • Collaborated with product designers in prototype debugging and performance evaluation, involving read/write lower-level registers mapping to hardware (HW), verification using signal generators/analyzers, power meters, and numerically-controlled oscillators to support the generation of continuous wave signals and 5G signals for specific NR bands, RF ports, and ASICs.
  • Documented RF test solutions for radio products, utilizing C++ file manipulations with the fstream library and C++ static methods. Implemented additional human-readable logging stored in the device under test.

Anritsu

I joined Anritsu's Measurement Group - ME7834 Department - 5G Protocol Conformance Test(PCT) in August 2017 as a Software Engineer and I progressed to the role of Senior Software Engineer and Team Leader through a promotion.

Responsibilities

  • Led and supervised a team of six engineers, ensuring their mentorship and streamlining the onboarding process for new members. Pioneered the implementation of operational frameworks, notably introducing Agile methodologies and integrating version control systems like Git. Instituted effective practices such as regular standup meetings, introduced automation processes, and documented test results to enhance overall efficiency. Served as the key liaison for global managers, and conducted weekly meetings with customers, ensuring alignment on project objectives.
  • Developed software solutions and test cases according to 3GPP specifications TS 38.523-1 and requirements for 5G and 4G protocols using TTCN-3. Led Anritsu's achievement of 3GPP Approval for the World's First 5G New Radio Standalone Mode Carrier Aggregation Test. Collaborated with a key customer to verify the industry’s first 5G NR Protocol Conformance Test in the Sub-6 GHz Frequency Range.
  • Automated test case validation for GCF / PTCRB using Python. Verified new industry 5G test cases based on 3GPP TS 38.523–1 for RAN5, including NSA FR1 measurement test cases, NSA FR2 measurement test cases, and SA FR1 CA test cases, among others.
  • Analyzed the MAC, RLC, PDCP, and RRC layers using test equipment and trace loggers like Protocol Analyzer, QXDM, and QPST. Contributed to the development and verification of various 5G test cases involving different aspects such as NSA FR2 DTCH, RLC, PDCP, SA FR1 VONR, and collaboration with other partners
